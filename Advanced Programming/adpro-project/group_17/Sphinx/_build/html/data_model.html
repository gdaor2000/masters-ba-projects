Data Model Module¶

The data_model module is responsible for defining the data structures and models used in the project.

This module contains one class with the purpose of analyzing flight data.

plot_most_used_airplane_models(countries=None, top_n=None)¶

Plots the most used airplane models based on the number of routes they operate on.

This method filters the routes based on the countries provided and then counts the occurrences of each airplane model to find the most used ones. It then plots the top N most used models.

countrieslist, optional

A list of country names to filter the airports. If None, no country filter is applied.

top_nint, optional

The number of top airplane models to display in the plot. If None, all models are displayed.

None

This method does not return any value but shows a bar plot.
