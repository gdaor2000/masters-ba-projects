real_distances Module¶

The real_distances module is responsible for defining the data structures and models used in the project.

This module provides functions for performing mathematical calculations and working with data.

real_distances.add_airports_distances(gaivota: Gaivota)¶

Update the airports_distances dictionary with distances between all pairs of airports.

gaivota: Gaivota

An instance of the Gaivota class containing airport data.

real_distances.calculate_distance(lat1: float, lon1: float, lat2: float, lon2: float) → float¶

Calculate the distance between two points on the Earth’s surface using the Haversine formula.

lat1: float

Latitude of the first point in degrees.

lon1: float

Longitude of the first point in degrees.

lat2: float

Latitude of the second point in degrees.

lon2: float

Longitude of the second point in degrees.

distance: float

The distance between the two points in kilometers.

real_distances.calculate_distance_between_airports(source: str, destination: str) → float¶

Calculate the distance between two airports given their IATAs.

source: str

The IATA code of the source airport.

destination: str

The IATA code of the destination airport.

distance: float

The distance between the source and destination airports in kilometers.

real_distances.haversine_vectorized(lat1, lon1, lat2, lon2)¶

Calculate the great-circle distance between two points on the Earth’s surface given their longitudes and latitudes using the Haversine formula.

lon1float

Longitude of the first point in degrees.

lat2float

Latitude of the second point in degrees.

lon2float

Longitude of the second point in degrees.

distancefloat
